# DOCKER PHP
### SafetyPay

Descargar e instalar Docker ToolBox
```
$ https://www.docker.com/products/docker-toolbox
```

Running
```
$ docker-compose build
$ docker-compose up -d
```

Open (reemplazar "XXX" por 100, 101, 102, ... dependiendod de la VM)
```
http://192.168.99.XXX:8000/
```


Stop
```
$ docker-compose kill
```
